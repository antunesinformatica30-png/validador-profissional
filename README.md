# 🚀 Validador Profissional

Sistema de validação de cadastro com foco em UX. Inclui validação de email profissional, análise de força de senha, 
feedback em tempo real, confirmação de senha e interface responsiva com microinterações.

## 💻 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Simular um sistema de cadastro moderno com boas práticas de validação e experiência do usuário.
